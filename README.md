# ISPConfig Role - Ansible

This role install and configure the ensemble of tools for ISPConfig system.

## ISPConfig Version :
- 3.1.1

## Compatible OS

- Debian Jessie

## Example Playbook

```
- hosts: all
  roles:
   - ispconfig
```
